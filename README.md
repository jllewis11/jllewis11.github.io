### Prerequisites

- Node v18.4.0

### Installation and Run

Step 1: Download packages from package.json<br/>
`npm i`<br/>

Step 2: Start a local Server<br/>
`npm run start`<br/>

### Avaliable Scripts

Format: prettier<br/>
`npm run format`<br/>

Predeploys a build and deploys a build using gh-pages to github pages<br/>
`npm run deploy`<br/>
